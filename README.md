# 小朋友下樓梯

使用 Canvas API 重製的經典小遊戲，操控角色踩踏各種類型的階梯，盡可能往下前進並存活。

## Live Demo
https://cutfysh1101.github.io/NS-SHAFT/

## 技能樹
- Canvas API
- JavaScript ES6+ (Class、箭頭函數、模組化設計)
- Vue 3 (排行榜元件)
- SASS
- dat.GUI (開發時調整難度)
- TweenMax/GSAP (動畫效果)

## 主要功能
- 6 種階梯類型：普通、彈跳、左/右傳送帶、地刺、消失階梯
- 難度遞增系統，每 20 層自動提升難度
- 生命值系統與紅屏受傷特效
- 背景音樂與音效（踩踏、彈跳、受傷、死亡）
- 排行榜紀錄（Top 10）
- 支援鍵盤控制（WASD / 方向鍵）與虛擬搖桿
- RWD 響應式設計，支援桌面與行動裝置

## 開發動機
此專案為 Hahow「動畫互動網頁特效入門」課程的複習作品，透過實作經典遊戲來練習 Canvas 繪圖、物理模擬、碰撞偵測與遊戲狀態管理等技術
